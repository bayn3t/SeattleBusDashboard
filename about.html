About This Project

Seattle Metro Bus Stops Map - An interactive visualization of King County Metro's entire bus network.

About the Map

Our map shows over 6400 bus stops in the Seattle area. We designed it so that when zoomed out, the bus stop density is displayed as a hexagonal-pattern heatmap, but as you zoom in, you can see individual stops. Red hexagons represent high bus stop concentration, while green ones represent low density. By clicking on a dot/stop, you can see which lines it serves, accessibility, shelter, and more. There is also a search bar that lets users search for lines by route and displays the path they take. We also designed this website to be accessible to desktop and mobile users.

Why We Built This

Understanding public transit networks can be overwhelming. This map makes it easy to see:

  • Where bus service is concentrated in the city
  • How transit coverage varies by neighborhood
  • Details about any specific bus stop
  • Route information and accessibility options

Technology

Built with open-source tools that work in any modern web browser:

  • Leaflet.js - Interactive mapping library
  • H3.js - Hexagonal grid system for heatmap
  • OpenStreetMap - Free, open map data
  • Vanilla JavaScript - No heavy frameworks, fast and lightweight

The entire map runs in your browser with no server needed. This means it loads fast and works offline (after the first load).

Data Source

  • Source: King County Metro Transit Stop Database
  • Coverage: Seattle, King County, and surrounding areas
  • Total Stops: 6,422 active stops
  • Updated: December 2025

About the Team

This project was created as part of a Geography 458 course project at the University of Washington. The team worked to turn complex transit data into an intuitive, visually engaging map that helps people understand Seattle's bus network at a glance.

Features

  • Zoom and Pan - Explore the entire region or focus on specific neighborhoods
  • Search - Find stops by name, ID, or route number
  • Click for Details - See routes, shelters, and accessibility info
  • Accessibility Info - Check ADA details for each stop
  • Mobile Friendly - Works great on phones and tablets
  • Fast & Lightweight - No heavy downloads or server delays

Open Source

All the code for this project is available on GitHub. Feel free to explore, modify, and use it for your own projects. We believe in making transit data accessible to everyone.
